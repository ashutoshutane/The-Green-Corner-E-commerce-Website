The Green Corner – E-Commerce Plant Selling Website

A full-stack E-Commerce application developed for selling plants online.
The project includes:
✔ Spring Boot REST API
✔ Spring Data JPA
✔ PostgreSQL Database
✔ React / Vite Frontend (optional)
✔ Postman-tested APIs
✔ Fully functional cart system, user management & plant filtering

✔ Backend
->Spring Boot
->Spring Web
->Spring Data JPA (Hibernate)
->PostgreSql
->Maven
->Java 17+

✔ Frontend(optional)
React (Vite)
Axios for API calls
Tailwind / CSS

✔ Tools
Postman (API Testing)
VS Code / Eclipse
PgAdmin4
